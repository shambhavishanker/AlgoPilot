**Project Overview**
This project focuses on building a system that can automatically select the most suitable search algorithm for solving a given problem. Algorithms like BFS, DFS, and A* perform differently depending on the structure of the problem, So it is not always easy to choose the one.
The system generates maze-based problems, extracts important features, and uses a machine learning model to predict which algorithm will perform best. The goal of the project is to reduce the amount of trial and error that people have to do. We want to show how machine learning can help people make decisions when they are trying to solve problems. The project uses machine learning to support decision making in problem solving with search algorithms, like Breadth First Search, Depth First Search and A Star.
The AlgoPilot project is trying to solve this problem by using machine learning. It looks at things like how dense the maze's how open it is if there are a lot of dead ends and how far apart things are. Then it uses this information to predict which algorithm will work the best for that maze.
The system also explains why a particular algorithm is chosen and visualizes the final path. This makes it not only useful for solving problems but also helpful for students to understand how different algorithms behave in different situations.


**OBJECTIVES**
The main objectives of this project are:

• To compare how BFS, DFS, and A* perform on different mazes

• To understand how the structure of a maze affects algorithm performance

• To use machine learning to predict the best algorithm for a given maze

• To explain why a particular algorithm is chosen

• To show the path taken by the selected algorithm

• To help students understand these algorithms better.

**How the Project Runs**

1. The system generates a random maze or accepts user-defined input.

2.  Features are extracted from the maze.

3. The trained machine learning model predicts the best algorithm.

4. The selected algorithm (BFS, DFS, or A*) is executed.
   
5. The system displays:
-The maze
-Performance metrics (time, nodes explored)
-Predicted algorithm
-Path visualization

**How to Run**
-Install dependencies
-Run the project: python main.py

**Conclusion**
This project demonstrates how machine learning can be used in a simple but effective way to improve algorithm selection. It combines search algorithms with data-driven decision-making to create a system that learns from experience and adapts over time.
